# Demo Script

1. Show the privacy warning and free-tier mode.
2. Create a workspace.
3. Upload public demo documents.
4. Watch ingestion status progress to ready.
5. Ask an answerable question in Fast Mode.
6. Open citations and evidence trace.
7. Ask a no-evidence question and show refusal.
8. Ask a conflicting-source question and show contradiction handling.
9. Run the evaluation dashboard.
10. Show docs and local quality-gate results. GitHub Actions are deferred until final hardening.
